# FolsomChat

The digital home for the Folsom, California community.

## Tech Stack

- [Next.js](https://nextjs.org/) (App Router)
- [React](https://react.dev/)
- [TypeScript](https://www.typescriptlang.org/)
- [Tailwind CSS](https://tailwindcss.com/)

## Getting Started

This project is fully configured for development in the cloud — no local Node.js installation required.

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the site.

## Production Build

```bash
npm run build
npm start
```

## Deploy on Vercel

The easiest way to deploy is through [Vercel](https://vercel.com):

1. Push this repository to GitHub
2. Import the repository at [vercel.com/new](https://vercel.com/new)
3. Vercel auto-detects Next.js — click Deploy

## Project Structure

```
app/              # App Router pages and layouts
components/       # Reusable UI components
lib/              # Utilities and shared config
public/           # Static assets
```
